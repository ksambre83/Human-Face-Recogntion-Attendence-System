# **App Name**: AttendAI

## Core Features:

- Teacher Authentication: Secure teacher login with ID, password, remember me, and forgot password functionality. Features a clean, attractive gradient background with animated shapes.
- Interactive Dashboard: Personalized teacher dashboard displaying key metrics like total students, daily presence/absence, attendance percentage, trend graphs, recent activity, defaulter lists, and quick actions, all within white dashboard cards.
- Attendance Module Setup: Allows teachers to select subject, class, division, and date, preparing for a live attendance session with a prominent 'Take Attendance' button.
- Live Face Recognition Attendance: Visually impressive page featuring a large, glassmorphism-styled camera card for a live webcam stream (http://127.0.0.1:5000/video_feed). Displays a real-time recognized students list with present badges and attendance counts, facilitated by the face recognition backend.
- Comprehensive Reports: Enables viewing and filtering of attendance data through a modern ERP-style table, with options to search by student, date, and subject, alongside export functionality to CSV and Excel, including pagination.
- Attendance Analytics: Provides insightful visualizations including attendance bar charts, pie charts, monthly trend graphs, defaulter charts, and subject-wise attendance graphs to analyze performance.

## Style Guidelines:

- Primary brand color for actions and highlights: A professional and vibrant green (#2CAD5C), ensuring good contrast against light backgrounds.
- Main application background: A very pale green (#F4FFEF) derived from the primary hue, providing a clean and subtle base.
- Accent color: A soft, slightly warmer chartreuse green (#C7EE8B), analogous to the primary, for secondary emphasis or graphical elements.
- Structural UI color: A deep dark navy blue (#1A284D) for the left sidebar and prominent UI elements, establishing a premium, institutional feel.
- Body and headline font: 'Inter' (sans-serif) for its modern, neutral, and highly readable characteristics, fitting an ERP dashboard's professional requirement.
- Use modern, professional vector icons with a consistent, clean-line style suitable for an institutional ERP, reflecting a minimalist yet informative aesthetic.
- Standard dashboard layout featuring a fixed dark navy blue left sidebar for navigation and a main content area comprising clean white dashboard cards. Optimized for responsive viewing on laptop and tablet devices.
- Implement clean animations for background shapes on the login page, smooth transitions for navigation elements and data updates, subtle hover effects, and crisp feedback for primary actions.
